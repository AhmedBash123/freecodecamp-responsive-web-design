Introduction

JavaScript is a programming language used to create interactive effects within web browsers.

It allows dynamic updates, user interactions, and more.

console.log("Hello, world!");
  • Web development
  • Frontend scripting
Variables

Variables store data values that can be reused throughout a program.

Use const for constants and let for values that can change.

let name = "Ahmed";
  • let
  • const
Functions

Functions group reusable blocks of code to perform specific tasks.

They can take parameters and return values.

function greet() { alert("Hi!"); }
  • Reusable code
  • Return values
Loops

Loops repeat a block of code until a condition is met.

They help automate repetitive tasks.

for (let i = 0; i < 5; i++) { console.log(i); }
  • for loop
  • while loop
Objects

Objects hold collections of data and related functionality.

Each property is a key-value pair.

let user = {name: "Ahmed", age: 25};
